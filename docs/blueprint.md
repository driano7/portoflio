# **App Name**: Donovan Riaño Portfolio

## Core Features:

- Landing Page: Display a landing page with a welcome message and the name 'Donovan Riaño'.
- About Me Section: Implement an 'About Me' section.
- GitHub Repositories Display: Automatically fetch and display public GitHub repositories from https://github.com/driano7?tab=repositories.
- Featured Projects: Feature projects prominently at the top: Strawberry Wallet (https://github.com/driano7/strawberry-wallet) and Xoco Café (https://github.com/driano7/xoco-cafe).
- Tech Stack Section: Show a 'Tech Stack' section, similar to https://github.com/yujisatojr/react-portfolio-template, displaying logos and descriptions for React, TypeScript, TailwindCSS, and Firebase.
- Theme Switcher: Allow users to switch between Light, Dark, and Dim themes via a visible selector.
- Footer with Social Links: Display a footer with links to social media profiles, including GitHub.

## Style Guidelines:

- Primary color: #0070f3 (Chronark's primary blue), conveying trust and innovation.
- Background color: #fff (white) for light mode and #111 (dark gray) for dark mode, ensuring readability and visual comfort.
- Accent color: #3291ff (a lighter blue), creating visual interest and highlighting interactive elements.
- Headline and body font: 'Lato' (sans-serif) for a clean, modern, and readable design.  Served from Google Fonts.
- Minimalist, clean, and responsive layout, similar to https://chronark.com, ensuring optimal viewing experience across devices.
- Use simple, geometric icons for the Tech Stack section, maintaining a consistent and modern aesthetic.
- Subtle transitions and hover effects to enhance interactivity without being distracting.
